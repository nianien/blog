4.4 Code ReviewпјҡдёӨж®өејҸе®ЎжҹҘ

иҮӘеҠЁеҢ–иғҪжӢҰдҪҸжЁЎејҸеҢ–зҡ„й—®йўҳпјҢдҪҶиҜӯд№үеұӮйқўзҡ„й”ҷиҜҜеҸӘжңүдәәиғҪеҸ‘зҺ°гҖӮ

ж ҮеҮҶпјҡ

  • Review йҮҚзӮ№дёҚжҳҜд»Јз ҒйЈҺж јпјҢиҖҢжҳҜеӣӣдёӘж ёеҝғз»ҙеәҰпјҡ
з»ҙеәҰ е…іжіЁзӮ№
еҘ‘зәҰе®Ңж•ҙжҖ§ иҫ“е…Ҙиҫ“еҮәжҳҜеҗҰз¬ҰеҗҲйў„жңҹпјҹжҺҘеҸЈжҳҜеҗҰеҗ‘еҗҺе…је®№пјҹ
й”ҷиҜҜеӨ„зҗҶ ејӮеёёи·Ҝеҫ„жҳҜеҗҰе®ҢеӨҮпјҹйҮҚиҜ•е’Ңе№ӮзӯүжҳҜеҗҰжӯЈзЎ®пјҹ
иө„жәҗиҫ№з•Ң еҶ…еӯҳгҖҒиҝһжҺҘж•°гҖҒе№¶еҸ‘жҳҜеҗҰжңүдёҠйҷҗпјҹtimeout жҳҜеҗҰеҗҲзҗҶпјҹ
е®үе…ЁжҖ§ иҫ“е…Ҙж ЎйӘҢжҳҜеҗҰе……еҲҶпјҹжҳҜеҗҰеӯҳеңЁжіЁе…ҘзӮ№пјҹж—Ҙеҝ—жҳҜеҗҰжі„жјҸж•Ҹж„ҹдҝЎжҒҜпјҹ

иҗҪең°ж–№ејҸпјҡ GitHub CODEOWNERS + Branch Protection RulesпјҢзЎ®дҝқ AI-assisted PR еҝ…йЎ»з»ҸиҝҮ review жүҚиғҪ mergeгҖӮ

5.1 Feature Flag + Canary ж”ҫйҮҸ

  • Canary ж”ҫйҮҸжўҜеәҰпјҡ1% вҶ’ 10% вҶ’ 50% вҶ’ 100%пјҢжҜҸдёҖжӯҘеҝ…йЎ»ж»Ўи¶і SLO жүҚиғҪз»§з»ӯ

5.2 иҮӘеҠЁеӣһж»ҡ

ж”ҫйҮҸиҝҮзЁӢдёӯпјҢд»ҘдёӢд»»дёҖжқЎд»¶и§ҰеҸ‘ж—¶еә”иҮӘеҠЁеӣһж»ҡпјҡ

жҢҮж Ү и§ҰеҸ‘жқЎд»¶
й”ҷиҜҜзҺҮ и¶…иҝҮеҹәзәҝ X%пјҲжҢүдёҡеҠЎе®ҡд№үпјү

5.3 еӣһж»ҡж–№жЎҲеҝ…йЎ»"еҸҜж“ҚдҪң"

еӣһж»ҡж–№ејҸ йҖӮз”ЁеңәжҷҜ жҒўеӨҚж—¶й—ҙ
е…ій—ӯ Feature Flag зәҜйҖ»иҫ‘еҸҳжӣҙпјҢж— зҠ¶жҖҒеҪұе“Қ з§’зә§

и“қз»ҝеҲҮжҚў еҹәзЎҖи®ҫж–ҪеҸҳжӣҙ еҲҶй’ҹзә§

6. зү№ж®ҠеңәжҷҜпјҡPipeline зұ»зі»з»ҹзҡ„йўқеӨ–и§„еҲҷ

иҝҷзұ»зі»з»ҹзҡ„ж ёеҝғйЈҺйҷ©жҳҜпјҡйҖ»иҫ‘еҸҳдәҶпјҢдҪҶзј“еӯҳжІЎеӨұж•ҲпјҢдҝ®ж”№еҗҺзҡ„д»Јз Ғж №жң¬дёҚдјҡиў«жү§иЎҢгҖӮ

6.1 йҖ»иҫ‘зүҲжң¬еҢ–

ж ҮеҮҶпјҡ д»»дҪ•еҪұе“ҚеӨ„зҗҶйҳ¶ж®өиҫ“еҮәиҜӯд№үзҡ„ж”№еҠЁпјҲз®—жі•гҖҒеӨ„зҗҶйҖ»иҫ‘гҖҒй»ҳи®ӨиЎҢдёәпјүпјҢеҝ…йЎ» bump phase.versionгҖӮ

class TranslationPhase(Phase):
    VERSION = "2026-02-15.1"  # иҜӯд№үеҸҳжӣҙж—¶еҝ…йЎ» bump

    def should_run(self, manifest):

def config_fingerprint(phase_name: str, config: dict) -> str:
    """еҜ№йҳ¶ж®өз”ҹж•Ҳй…ҚзҪ®еҒҡзЁіе®ҡеәҸеҲ—еҢ–еҗҺеҸ– hash"""
    effective = get_effective_config(phase_name, config)
    serialized = json.dumps(effective, sort_keys=True)
    return hashlib.sha256(serialized.encode()).hexdigest()[:16]

8. жҖ»з»“

AI зј–зЁӢе·Ҙе…·зҡ„з”ҹдә§еҠӣд»·еҖјжҜӢеәёзҪ®з–‘гҖӮдҪҶ**"и®© AI еҶҷд»Јз Ғ"е’Ң"и®© AI д»Јз ҒдёҠз”ҹдә§"д№Ӣй—ҙпјҢйңҖиҰҒдёҖж•ҙеҘ—е·ҘзЁӢжңәеҲ¶жқҘеЎ«иЎҘ**гҖӮ

иҝҷеҘ—жңәеҲ¶зҡ„ж ёеҝғйҖ»иҫ‘пјҡ

  • з”ҹжҲҗж—¶зәҰжқҹпјҡйҖҡиҝҮеҘ‘зәҰе…ҲиЎҢгҖҒTests-First е’ҢзҰҒжӯўжё…еҚ•пјҢд»ҺжәҗеӨҙйҷҚдҪҺдёҚеҗҲж јд»Јз Ғзҡ„дә§еҮәжҰӮзҺҮ
  • еҗҲе№¶ж—¶жӢҰжҲӘпјҡйҖҡиҝҮ CI GateгҖҒеҚұйҷ©жЁЎејҸжү«жҸҸе’Ңз»“жһ„еҢ– ReviewпјҢи®©дёҚеҗҲж јд»Јз Ғж— жі•иҝӣе…Ҙдё»е№І
  • еҸ‘еёғж—¶е…ңеә•пјҡйҖҡиҝҮ Feature FlagгҖҒCanary ж”ҫйҮҸе’ҢиҮӘеҠЁеӣһж»ҡпјҢеҚідҪҝжңүжјҸзҪ‘д№Ӣйұјд№ҹиғҪеҝ«йҖҹжӯўжҚҹ

AI дёҚзЎ®е®ҡжҖ§зҡ„жң¬иҙЁжҳҜпјҡдҪ ж— жі•еңЁз”ҹжҲҗйҳ¶ж®өж¶ҲзҒӯжүҖжңүйЈҺйҷ©гҖӮ жүҖд»Ҙзӯ”жЎҲдёҚжҳҜ"еҶҷжӣҙеҘҪзҡ„ prompt"пјҢиҖҢжҳҜ"е»әжӣҙеҘҪзҡ„е·ҘзЁӢйҳІзәҝ"гҖӮ

жҠҠжҜҸдёҖж¬Ў AI зҠҜзҡ„й”ҷзј–з ҒжҲҗдёҖжқЎиҮӘеҠЁи§„еҲҷпјҢи®©йҳІзәҝйҡҸз»ҸйӘҢдёҖиө·з”ҹй•ҝвҖ”вҖ”иҝҷжүҚжҳҜдёҺ AI еҚҸдҪңзј–зЁӢзҡ„еҸҜжҢҒз»ӯж–№ејҸгҖӮ
