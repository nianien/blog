gRPC зҡ„ж ёеҝғдјҳеҠҝеңЁдәҺејәзұ»еһӢеҘ‘зәҰпјҲProtobufпјүе’Ңй«ҳж•Ҳзҡ„дәҢиҝӣеҲ¶дј иҫ“пјҲHTTP/2пјүгҖӮдҪҶеңЁе·ҘзЁӢиҗҪең°дёӯпјҢдёӨдёӘй—®йўҳеҫҖеҫҖеҶіе®ҡдәҶзі»з»ҹзҡ„еҸҜз»ҙжҠӨжҖ§пјҡеҰӮдҪ•з»ҹдёҖеӨ„зҗҶжЁӘеҲҮе…іжіЁзӮ№пјҲж—Ҙеҝ—гҖҒи®ӨиҜҒгҖҒжҢҮж Үпјүе’ҢеҰӮдҪ•и®ҫи®Ўжё…жҷ°зҡ„й”ҷиҜҜдј йҖ’жңәеҲ¶гҖӮжң¬ж–ҮиҒҡз„ҰиҝҷдёӨдёӘж ёеҝғй—®йўҳгҖӮ

дәҢгҖҒжӢҰжҲӘеҷЁжңәеҲ¶

2.1 жӢҰжҲӘеҷЁзҡ„е®ҡдҪҚ

gRPC жӢҰжҲӘеҷЁзӯүеҗҢдәҺ HTTP дё–з•Ңдёӯзҡ„ Filter / MiddlewareпјҢз”ЁдәҺеңЁ RPC и°ғз”Ёзҡ„еүҚеҗҺжҸ’е…ҘжЁӘеҲҮйҖ»иҫ‘пјҡ

  • иҜ·жұӮ/е“Қеә”ж—Ҙеҝ—и®°еҪ•
  • и®ӨиҜҒдёҺйүҙжқғпјҲToken ж ЎйӘҢгҖҒжқғйҷҗжЈҖжҹҘпјү
  • жҢҮж ҮйҮҮйӣҶпјҲи°ғз”ЁиҖ—ж—¶гҖҒй”ҷиҜҜзҺҮпјү
  • й“ҫи·ҜиҝҪиёӘпјҲTraceId дј йҖ’пјү
  • е…ғж•°жҚ®жіЁе…ҘпјҲиҜ·жұӮ IDгҖҒз§ҹжҲ·ж ҮиҜҶпјү

2.2 Client жӢҰжҲӘеҷЁ

е®ўжҲ·з«ҜжӢҰжҲӘеҷЁе®һзҺ° ClientInterceptor жҺҘеҸЈпјҢеңЁеҸ‘иө· RPC и°ғз”Ёж—¶д»Ӣе…ҘгҖӮ

public class LoggingClientInterceptor implements ClientInterceptor {
    @Override
    public <ReqT, RespT> ClientCall<ReqT, RespT> interceptCall(
            MethodDescriptor<ReqT, RespT> method,
            CallOptions callOptions,
            Channel next) {

        return new ForwardingClientCall.SimpleForwardingClientCall<>(
                next.newCall(method, callOptions)) {

            @Override
            public void start(Listener<RespT> responseListener, Metadata headers) {
                // иҜ·жұӮеҸ‘еҮәеүҚпјҡжіЁе…Ҙе…ғж•°жҚ®
                headers.put(REQUEST_ID_KEY, UUID.randomUUID().toString());

                super.start(new ForwardingClientCallListener
                        .SimpleForwardingClientCallListener<>(responseListener) {

                    @Override
                    public void onHeaders(Metadata headers) {
                        // ж”¶еҲ°е“Қеә”еӨҙ
                        super.onHeaders(headers);
                    }

                    @Override
                    public void onMessage(RespT message) {
                        // ж”¶еҲ°е“Қеә”ж¶ҲжҒҜ
                        super.onMessage(message);
                    }

                    @Override
                    public void onClose(Status status, Metadata trailers) {
                        // RPC з»“жқҹпјҡи®°еҪ•зҠ¶жҖҒ
                        log.info("{} completed with status: {}",
                                method.getFullMethodName(), status.getCode());
                        super.onClose(status, trailers);
                    }
                }, headers);
            }

            @Override
            public void sendMessage(ReqT message) {
                // еҸ‘йҖҒиҜ·жұӮж¶ҲжҒҜ
                super.sendMessage(message);
            }
        };
    }
}

е®ўжҲ·з«Ҝи°ғз”Ёй“ҫи·ҜпјҲUnary RPCпјүпјҡ

еә”з”Ёд»Јз Ғи°ғз”Ё stub ж–№жі•

    вҶ’ ForwardingClientCall.start()        [еҮәз«ҷпјҡи®ҫзҪ®е…ғж•°жҚ®]
    вҶ’ ForwardingClientCall.sendMessage()  [еҮәз«ҷпјҡеҸ‘йҖҒиҜ·жұӮ]
    вҶ’ ForwardingClientCall.halfClose()    [еҮәз«ҷпјҡиҜ·жұӮз»“жқҹ]
    вҶҗ CallListener.onHeaders()            [е…Ҙз«ҷпјҡж”¶еҲ°е“Қеә”еӨҙ]
    вҶҗ CallListener.onMessage()            [е…Ҙз«ҷпјҡж”¶еҲ°е“Қеә”дҪ“]
    вҶҗ CallListener.onClose()              [е…Ҙз«ҷпјҡRPC з»“жқҹ]

2.3 Server жӢҰжҲӘеҷЁ

жңҚеҠЎз«ҜжӢҰжҲӘеҷЁе®һзҺ° ServerInterceptor жҺҘеҸЈпјҢеңЁеӨ„зҗҶж”¶еҲ°зҡ„ RPC иҜ·жұӮж—¶д»Ӣе…ҘгҖӮ

public class AuthServerInterceptor implements ServerInterceptor {
    @Override
    public <ReqT, RespT> ServerCall.Listener<ReqT> interceptCall(
            ServerCall<ReqT, RespT> call,
            Metadata headers,
            ServerCallHandler<ReqT, RespT> next) {

        // 1. д»Һе…ғж•°жҚ®дёӯжҸҗеҸ–и®ӨиҜҒдҝЎжҒҜ
        String token = headers.get(AUTH_TOKEN_KEY);
        if (!isValid(token)) {
            call.close(Status.UNAUTHENTICATED
                    .withDescription("Invalid token"), new Metadata());
            return new ServerCall.Listener<>() {};  // иҝ”еӣһз©ә ListenerпјҢдёҚеӨ„зҗҶеҗҺз»ӯиҜ·жұӮ
        }

        // 2. еҢ…иЈ… ServerCall д»ҘжӢҰжҲӘе“Қеә”
        ServerCall<ReqT, RespT> wrappedCall = new ForwardingServerCall
                .SimpleForwardingServerCall<>(call) {

            @Override
            public void sendMessage(RespT message) {
                // жӢҰжҲӘе“Қеә”ж¶ҲжҒҜ
                super.sendMessage(message);
            }

            @Override
            public void close(Status status, Metadata trailers) {
                // RPC з»“жқҹж—¶зҡ„еӨ„зҗҶ
                super.close(status, trailers);
            }
        };

        // 3. еҢ…иЈ… Listener д»ҘжӢҰжҲӘиҜ·жұӮ
        ServerCall.Listener<ReqT> listener = next.startCall(wrappedCall, headers);

        return new ForwardingServerCallListener
                .SimpleForwardingServerCallListener<>(listener) {

            @Override
            public void onMessage(ReqT message) {
                // ж”¶еҲ°иҜ·жұӮж¶ҲжҒҜ
                super.onMessage(message);
            }

            @Override
            public void onHalfClose() {
                // е®ўжҲ·з«ҜеҸ‘йҖҒе®ҢжҜ•
                super.onHalfClose();
            }

            @Override
            public void onComplete() {
                // RPC е®ҢжҲҗ
                super.onComplete();
            }
        };
    }
}

ж”¶еҲ°е®ўжҲ·з«ҜиҜ·жұӮ
  вҶ’ ServerInterceptor.interceptCall()
    вҶҗ Listener.onMessage()          [е…Ҙз«ҷпјҡж”¶еҲ°иҜ·жұӮдҪ“]
    вҶҗ Listener.onHalfClose()        [е…Ҙз«ҷпјҡе®ўжҲ·з«ҜеҸ‘йҖҒе®ҢжҜ•]
    вҶ’ дёҡеҠЎйҖ»иҫ‘еӨ„зҗҶ
    вҶ’ ServerCall.sendHeaders()      [еҮәз«ҷпјҡеҸ‘йҖҒе“Қеә”еӨҙ]
    вҶ’ ServerCall.sendMessage()      [еҮәз«ҷпјҡеҸ‘йҖҒе“Қеә”дҪ“]
    вҶ’ ServerCall.close()            [еҮәз«ҷпјҡз»“жқҹ RPC]

3.2 дёӨз§Қй”ҷиҜҜжЁЎеһӢ

gRPC жҸҗдҫӣдәҶдёӨз§Қй”ҷиҜҜдј йҖ’жЁЎеһӢпјҢйҖӮз”ЁдәҺдёҚеҗҢзҡ„еӨҚжқӮеәҰйңҖжұӮпјҡ

жЁЎеһӢдёҖпјҡio.grpc.StatusпјҲеҹәзЎҖжЁЎеһӢпјү

йҖҡиҝҮ StatusRuntimeException жҗәеёҰзҠ¶жҖҒз Ғе’ҢжҸҸиҝ°дҝЎжҒҜгҖӮж”ҜжҢҒйҖҡиҝҮ Metadata йҷ„еҠ иҮӘе®ҡд№үй”ҷиҜҜиҜҰжғ…гҖӮ

// жңҚеҠЎз«Ҝпјҡиҝ”еӣһй”ҷиҜҜ
@Override
public void getPrice(PriceRequest request, StreamObserver<PriceResponse> observer) {
    if (request.getCommodity().isEmpty()) {
        // ж–№ејҸ 1пјҡд»…зҠ¶жҖҒз Ғ + жҸҸиҝ°
        observer.onError(Status.INVALID_ARGUMENT
                .withDescription("commodity cannot be empty")
                .asRuntimeException());
        return;
    }

    // ж–№ејҸ 2пјҡйҷ„еҠ иҮӘе®ҡд№үе…ғж•°жҚ®
    Metadata metadata = new Metadata();
    Metadata.Key<ErrorResponse> key = ProtoUtils.keyForProto(ErrorResponse.getDefaultInstance());
    metadata.put(key, ErrorResponse.newBuilder()
            .setCode("INVALID_COMMODITY")
            .setMessage("Commodity not found: " + request.getCommodity())
            .build());

    observer.onError(Status.NOT_FOUND
            .withDescription("Commodity not found")
            .asRuntimeException(metadata));
}

// е®ўжҲ·з«ҜпјҡжҸҗеҸ–й”ҷиҜҜ
try {
    PriceResponse response = stub.getPrice(request);
} catch (StatusRuntimeException e) {

    Metadata trailers = Status.trailersFromThrowable(e);

}

жҺЁиҚҗзӯ–з•ҘпјҡеҶ…йғЁеҫ®жңҚеҠЎз»ҹдёҖдҪҝз”Ё google.rpc.Status жЁЎеһӢпјҢиҺ·еҫ—з»“жһ„еҢ–зҡ„й”ҷиҜҜдҝЎжҒҜпјӣйқўеҗ‘еӨ–йғЁзҡ„ API дҪҝз”Ё io.grpc.Status жЁЎеһӢпјҢдҝқиҜҒе…је®№жҖ§гҖӮ

3.4 жөҒејҸ RPC зҡ„й”ҷиҜҜеӨ„зҗҶ

еңЁжөҒејҸ RPC дёӯпјҢonError() жҳҜз»ҲжӯўжҖ§ж“ҚдҪңвҖ”вҖ”и°ғз”ЁеҗҺиҝһжҺҘз«ӢеҚіж–ӯејҖпјҢеҗҺз»ӯж¶ҲжҒҜж— жі•еҸ‘йҖҒгҖӮеӣ жӯӨпјҢжөҒејҸеңәжҷҜдёӢзҡ„й”ҷиҜҜдёҚеә”йҖҡиҝҮ onError() дј йҖ’пјҢиҖҢеә”еөҢе…ҘеҲ°ж¶ҲжҒҜдҪ“дёӯгҖӮ

// еңЁж¶ҲжҒҜе®ҡд№үдёӯдҪҝз”Ё oneof жҗәеёҰжӯЈеёёж•°жҚ®жҲ–й”ҷиҜҜдҝЎжҒҜ
message StreamingResponse {
    oneof payload {
        DataMessage data = 1;
        google.rpc.Status error = 2;
    }
}

// жңҚеҠЎз«ҜпјҡеңЁжөҒдёӯеҸ‘йҖҒй”ҷиҜҜпјҲдёҚдёӯж–ӯжөҒпјү
@Override
public void streamPrices(PriceRequest request,
        StreamObserver<StreamingResponse> observer) {
    for (String commodity : commodities) {
        try {
            DataMessage data = fetchPrice(commodity);
            observer.onNext(StreamingResponse.newBuilder()
                    .setData(data).build());
        } catch (Exception e) {
            // й”ҷиҜҜеөҢе…Ҙж¶ҲжҒҜдҪ“пјҢжөҒдёҚдёӯж–ӯ
            observer.onNext(StreamingResponse.newBuilder()
                    .setError(com.google.rpc.Status.newBuilder()
                            .setCode(Code.INTERNAL.getNumber())
                            .setMessage(e.getMessage())
                            .build())
                    .build());
        }
    }
    observer.onCompleted();  // жӯЈеёёз»“жқҹжөҒ
}

еӣӣгҖҒз”ҹдә§зә§жңҖдҪіе®һи·ө

4.1 и¶…ж—¶дёҺ Deadline

gRPC дҪҝз”Ё Deadline иҖҢйқһ Timeout жқҘжҺ§еҲ¶и¶…ж—¶гҖӮDeadline жҳҜдёҖдёӘз»қеҜ№ж—¶й—ҙзӮ№пјҢеңЁи°ғз”Ёй“ҫдёӯиҮӘеҠЁдј йҖ’е’ҢйҖ’еҮҸгҖӮ

// и®ҫзҪ® Deadline
PriceResponse response = stub
    .withDeadlineAfter(500, TimeUnit.MILLISECONDS)
    .getPrice(request);

Deadline дј ж’ӯпјҡеҪ“ Service A и°ғз”Ё Service BпјҢService B еҶҚи°ғз”Ё Service C ж—¶пјҢDeadline дјҡиҮӘеҠЁдј йҖ’гҖӮеҰӮжһң A и®ҫзҪ®дәҶ 500ms DeadlineпјҢз»ҸиҝҮ AвҶ’B иҖ—ж—¶ 200msпјҢBвҶ’C зҡ„ Deadline иҮӘеҠЁеҸҳдёә 300msгҖӮ

4.2 йҮҚиҜ•й…ҚзҪ®

gRPC ж”ҜжҢҒеңЁжңҚеҠЎй…ҚзҪ®дёӯеЈ°жҳҺйҮҚиҜ•зӯ–з•Ҙпјҡ

  "methodConfig": [{

д»…еҜ№е№Ӯзӯүж“ҚдҪңй…ҚзҪ®йҮҚиҜ•гҖӮйқһе№Ӯзӯүж“ҚдҪңпјҲеҰӮеҲӣе»әи®ўеҚ•пјүдёҚеә”иҮӘеҠЁйҮҚиҜ•гҖӮ

4.3 е…ғж•°жҚ®дј йҖ’жЁЎејҸ

йҖҡиҝҮжӢҰжҲӘеҷЁз»ҹдёҖжіЁе…Ҙе’ҢжҸҗеҸ–е…ғж•°жҚ®пјҡ

// е®ҡд№үе…ғж•°жҚ® Key

4.4 жӢҰжҲӘеҷЁжү§иЎҢйЎәеәҸ

еӨҡдёӘжӢҰжҲӘеҷЁеҪўжҲҗй“ҫејҸи°ғз”ЁгҖӮзҗҶи§Јжү§иЎҢйЎәеәҸеҜ№дәҺи°ғиҜ•иҮіе…ійҮҚиҰҒпјҡ

жіЁеҶҢйЎәеәҸпјҡinterceptor A, interceptor B

Client з«Ҝжү§иЎҢйЎәеәҸпјҲLIFOпјүпјҡ
  еҮәз«ҷиҜ·жұӮпјҡB вҶ’ A вҶ’ зҪ‘з»ң
  е…Ҙз«ҷе“Қеә”пјҡA вҶ’ B вҶ’ еә”з”Ё

Server з«Ҝжү§иЎҢйЎәеәҸпјҲFIFOпјүпјҡ
  е…Ҙз«ҷиҜ·жұӮпјҡA вҶ’ B вҶ’ дёҡеҠЎйҖ»иҫ‘
  еҮәз«ҷе“Қеә”пјҡдёҡеҠЎйҖ»иҫ‘ вҶ’ B вҶ’ A вҶ’ зҪ‘з»ң

е»әи®®е°Ҷи®ӨиҜҒжӢҰжҲӘеҷЁж”ҫеңЁжңҖеүҚйқўпјҲжңҖе…Ҳжү§иЎҢпјүпјҢж—Ҙеҝ—жӢҰжҲӘеҷЁж”ҫеңЁжңҖеҗҺйқўпјҲеҢ…иЈ№жүҖжңүйҖ»иҫ‘пјүгҖӮ

жҖ»з»“

gRPC е·ҘзЁӢеҢ–зҡ„дёӨдёӘж ёеҝғй—®йўҳвҖ”вҖ”жӢҰжҲӘеҷЁе’Ңй”ҷиҜҜеӨ„зҗҶвҖ”вҖ”еҶіе®ҡдәҶзі»з»ҹзҡ„еҸҜи§ӮжөӢжҖ§е’ҢеҸҜз»ҙжҠӨжҖ§пјҡ

  1. жӢҰжҲӘеҷЁжҳҜ gRPC зҡ„жЁӘеҲҮе…іжіЁзӮ№еҹәзЎҖи®ҫж–ҪгҖӮзҗҶи§Ј ForwardingClientCall / ForwardingServerCall еҸҠе…¶ Listener зҡ„еҸҢеҗ‘и°ғз”Ёй“ҫи·ҜпјҢжҳҜжӯЈзЎ®е®һзҺ°ж—Ҙеҝ—гҖҒи®ӨиҜҒгҖҒй“ҫи·ҜиҝҪиёӘзҡ„еүҚжҸҗ
  2. й”ҷиҜҜеӨ„зҗҶйңҖиҰҒеҢәеҲҶ Unary е’Ң StreamingгҖӮUnary и°ғз”ЁдҪҝз”Ё onError() иҝ”еӣһй”ҷиҜҜзҠ¶жҖҒпјӣжөҒејҸи°ғз”Ёеә”е°Ҷй”ҷиҜҜеөҢе…Ҙж¶ҲжҒҜдҪ“пјҢйҒҝе…Қдёӯж–ӯж•°жҚ®жөҒ
  3. дјҳе…ҲдҪҝз”Ё google.rpc.Status жЁЎеһӢгҖӮйў„е®ҡд№үзҡ„ ErrorInfoгҖҒRetryInfo зӯүзұ»еһӢжҸҗдҫӣдәҶз»“жһ„еҢ–зҡ„й”ҷиҜҜдҝЎжҒҜпјҢжҜ”иҮӘе®ҡд№ү Metadata жӣҙи§„иҢғ

gRPC зҡ„ API и®ҫи®ЎзІҫз®ҖдҪҶжҠҪиұЎзЁӢеәҰй«ҳгҖӮеңЁз”ҹдә§зҺҜеўғдёӯпјҢжӢҰжҲӘеҷЁе’Ңй”ҷиҜҜеӨ„зҗҶзҡ„жЁЎејҸеҢ–е®һзҺ°пјҢжҜ”жҜҸдёӘжңҚеҠЎзҡ„йҖҗдёҖеӨ„зҗҶжӣҙеҸҜйқ гҖҒжӣҙеҸҜз»ҙжҠӨгҖӮ
